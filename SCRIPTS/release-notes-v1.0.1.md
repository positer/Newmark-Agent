# Newmark Agent v1.0.1

Newmark Agent v1.0.1 is the first public release candidate of the local-first desktop Agent terminal.

## Download

- `Newmark-Agent-1.0.1-portable-x64.exe`
- SHA256: `8d84f5984cd969a03d8f435bf0e19d6ef2ae765dc2b1d590b0fb88e75cec5d65`

## Highlights

- Windows portable desktop app.
- Build, Plan, Goal, and Flow modes.
- Workspace-scoped conversations, plans, and archives.
- Subagents, skills, browser tools, GitHub CLI tooling, and automation tools.
- OpenAI-compatible and Anthropic-compatible provider configuration.
- Packaged CLI and packaged UI validation for the release path.

## Status

Release judgment: `release-usable-with-operational-risks`.

Remaining hardening areas include code signing, installer distribution, broader field monitoring, and public CI/release automation.
