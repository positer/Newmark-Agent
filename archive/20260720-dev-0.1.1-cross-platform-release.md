# dev-0.1.1 Cross-Platform Release

Date: 2026-07-20

## Scope

Final source, test, Windows package, Linux package, Git, and GitHub prerelease publication for `dev-0.1.1`.

## Final Maintenance Additions

- Persisted crashed/force-closed `running` Build records recover as `interrupted` with frozen duration.
- Context compression atomically replaces live Kernel context and suppresses repeated same-baseline compression.
- Compression triggers at 80% of the active model context window and targets 20%; fixed character thresholds no longer trigger early compression.
- Applicable historical unfinished tasks are resumed newest-to-oldest.
- Model validation reads explicit context-window fields from provider catalog responses and persists valid limits while ignoring output-only token fields.
